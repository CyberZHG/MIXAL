◆ ExprParseState

enum mixal::ExprParseState
strong

The states while parsing the expression.

EXPRESSION -> + EXPRESSION
| - EXPRESSION
| EXPRESSION OPERATOR EXPRESSION
| ATOMIC
OPERATOR -> + | - | * | / | // | :
ATOMIC -> INTEGER
| SYMBOL
| *
INTEGER -> DIGIT
| DIGIT INTEGER
SYMBOL -> ALPHA
| DIGIT
| ALPHS SYMBOL
| DIGIT SYMBOL
DIGIT -> [0-9]
ALPHA -> [A-Z]

However, multiple signs like ++ or --- will be rejected as they are useless.
